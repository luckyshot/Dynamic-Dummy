DynamicDummy
===================

**Generate random placeholders to test how variation affects an HTML design**

Methods
-------------------

### Color
<pre>$d->color();</pre>
Returns an color: <code>#FB7A20</code>.

### Words
<pre>$d->words(2, 6);</pre>
Returns a sentence: <code>Learn never since</code>.

### Weekday
<pre>$d->weekday();</pre>
Returns a day of the week: <code>thu</code>.
